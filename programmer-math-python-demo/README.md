# 《程序员学习：用Python学透线性代数和微积分》随书代码

本书旨在帮助程序员朋友们通过Python语言，深入浅出地学习线性代数和微积分的核心概念，并将其应用于实际的编程工作中。

本仓库为书籍配套的演示代码。

## 目录结构

- `linear_algebra`: 线性代数相关代码
  - `01_vectors`: 向量
  - `02_matrices`: 矩阵
  - `03_linear_systems`: 线性方程组
  - `04_eigenvalues_eigenvectors`: 特征值和特征向量
- `calculus`: 微积分相关代码
  - `01_limits`: 极限
  - `02_derivatives`: 导数
  - `03_integrals`: 积分
  - `04_differential_equations`: 微分方程
